== Installation ==

Please ensure that http://wordpress.org/plugins/syntaxhighlighter/ is installed

1. Upload `wp-` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the `[arduino][/arduino]` or `[code lang="arduino"][/code]` shortcodes to embed arduino code.
